# LibreView-CSV-Analasys

Important! This code is in early testing, data highly innacurate

# Info:
This project is supposed to help analyze the data given by "https://www.libreview.com"s CSV file,
and try to calculate the amount of insulin needed per KH depending on time of day.

This isn’t meant to replace professional advice, but a tool to help those who need it. always consider consulting a doctor before making any decisions about your health.


# How do use Program:
IMPORTANT! In order for this program to give any relevant data, you NEED to use the LibreLinks note function regulearly to note down Insulin, and Carbs taken.
Download your CSV file from "https://www.libreview.com", and paste it into the data folder.
Then modify main.py's 'settings' at the top of the file and run it
